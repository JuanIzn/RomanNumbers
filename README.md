# Roman Numerals Converter

This is a simple university project to convert integers to Roman numerals and vice-versa.

## Features
- Integer to Roman conversion (1 - 3999).
- Roman to Integer conversion (supports case-insensitivity).
- Simple and responsive web interface.

## How to use
Simply open the `index.html` file in any web browser.

## Testing
To run the automated tests in the terminal:
```bash
npm test
```

Tests are automatically executed on every push via GitHub Actions.
